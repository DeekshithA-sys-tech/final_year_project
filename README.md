# Tuberculosis Detection from Chest X-Ray using Deep Learning

## 📌 Project Overview
This project is a Final Year Machine Learning project that detects **Tuberculosis (TB)** from **Chest X-Ray images** using Deep Learning models such as ResNet50, DenseNet, and GAN-based enhancement methods.

The system provides a complete solution with:

- ML Model Prediction
- Backend API
- Frontend Web Application
- User-friendly interface

---

## ❗ Problem Statement
Tuberculosis is a serious lung disease that requires early diagnosis. Manual screening of X-rays takes time and depends on radiologist expertise.

This project automates TB detection using AI to provide faster and more accurate results.

---

## 🚀 Features

✅ Upload Chest X-Ray image  
✅ Predict TB Positive / Negative  
✅ Deep Learning Models Used  
✅ Frontend + Backend Integration  
✅ Fast and Accurate Results
